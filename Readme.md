# Astrus OS 🚀

Astrus OS é um sistema operacional experimental e moderno criado com foco em performance, personalização e aprendizado.  
Este projeto faz parte do ecossistema Astrus e possui uma versão **open source** para colaboração da comunidade.

---

## 🌍 Sobre o projeto

O Astrus OS tem como objetivo criar um sistema operacional leve, rápido e modular, permitindo que desenvolvedores possam estudar, modificar e contribuir livremente com o código.

---

## ⚙️ Objetivos

- Criar um sistema leve e rápido
- Permitir customização total do usuário
- Estimular aprendizado em sistemas operacionais
- Desenvolver um ecossistema próprio da Astrus

---

## 🧩 Sistema de Versões

O Astrus OS utiliza um sistema de versões próprio inspirado em sistemas modernos como Windows e macOS.

### 📌 Estrutura

XXYYZ

Exemplo:

EC28G

### 🔍 Significado

- **XX** → Nome/Geração do sistema
- **YY** → Ano ou geração principal
- **Z** → Edição do sistema

## 🔧 Builds

As builds internas do Astrus OS são separadas da versão principal e utilizam um sistema baseado em:

Bimestre → Mês → Dia

### 📌 Estrutura

B#.MM.DD

### 🔍 Significado

- **B#** → Bimestre da build
- **MM** → Mês da criação/lançamento
- **DD** → Dia da criação/lançamento

---

## 💡 Exemplos

| Código | Significado |
|---|---|
| B1.01.12 | 1º bimestre, Janeiro, dia 12 |
| B2.03.28 | 2º bimestre, Março, dia 28 |
| B4.08.05 | 4º bimestre, Agosto, dia 05 |
| B6.12.30 | 6º bimestre, Dezembro, dia 30 |
